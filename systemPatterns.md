# System Patterns

## Architecture

- **Frontend**: React (Vite) + Tailwind CSS.
- **Backend**: Firebase (Serverless).
- **State Management**: React Hooks (useState, useMemo) + Firebase Realtime Listeners.

## Design Patterns

- **Floating Island Navigation**: A persistent, floating nav bar.
- **Glassmorphism**: Usage of `backdrop-filter: blur` and semi-transparent backgrounds.
- **Component Structure**: Atomic design principles (though currently distinct in a single file for demo, mapped to Components).

## Tech Stack Rules

- **Styling**: Tailwind CSS is preferred, augmented by custom `style` tags for dynamic/complex effects if needed (GPU optimized).
- **Icons**: `lucide-react`.
